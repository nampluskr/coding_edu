## 제곱근

### 문제


임의의 정수 N이 주어졌을 때 N의 양의 제곱근의 정수부분을 출력하는 프로그램을 작성하라.

양의 제곱근이란 다음을 만족하는 수 X 를 뜻한다.



$$N = X^2\quad (X≥1)$$


**[ 주의  !!! ]**

sqrt와 같은 함수를 사용하지 말아야 하며

stdio.h 와 iostream 등 입출력 헤더에 있는 함수만이 사용가능하다.

python도 반칙은 안되니 출제 의의에 맞게 풀자.

이를 어길 경우 0점 처리한다.


### 입력
입력에는 263-1 이하의 양의 정수 N이 입력된다


### 출력
N의 제곱근의 정수부분을 출력한다.


### 예제1
입력
```
8
```

출력
```
2
```

### 예제2
입력
```
16
```

출력
```
4
```
